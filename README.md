# tibame_group_project

## 前置作業
1. 下載此包專案
```
git clone https://github.com/KakaCheng/tibame_group_project.git
```
2. 於根目錄下建立/models資料夾
    - 放置**haarcascade_frontalface_default.xml**
    - 放置**model.h5**\
(來源載點請參考\[1]說明)

3. 安裝套件
```
pip install -r requirements.txt
```
